>Given an integer array of size n, find all elements that appear more than âŒŠ n/3 âŒ‹ times.
>
>>Example 1:  
>>
>>Input: nums = [3,2,3]  
>>Output: [3]  

```python
class Solution:
    def majorityElement(self, nums):
        return [num for num, count in Counter(nums).items() if count > len(nums) // 3]
```
